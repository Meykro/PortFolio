Installation et configuration d'un serveur Observium


Besoin: Instalation d'un service de monitoring nommé Observium


Problématique: L'ephad de Moirans n'a pas de système de monitoring

    Réalisation:

    J'ai donc installé cette solution sur un serveur virtuel sous Ubuntu Server 22.04.

    Observium se configure sur l'interface Web, pour communiquer avec les appareils, il utilise le protocole SNMP (Simple Network Management Protocole) V1 qui est devenue obseléte car il n'est plus assez sécurisé, V2 le V2c ou le V3.
    J'ai personnellement opté pour la version V3 car il faut un login et un MDP pour ajoutées une machine sur Observium

    Ensuite j'ai du rajouter chaque machines du réseaux Moirans.
    Pour finir j'ai configuré des alertes, cela permet qu'en cas de pannes ou arrêt impromptu nous soyons prévenus.

    J’ai rencontré deux difficultés majeures lors de cette activité.
    La première:
    Les tutoriels pour configurer Observium sont souvent obselètes ou pas assez complets, donc cela ajoute de la difficultée a la mise en service.
    La deuxième:
    J'ai rencontré une erreur que je n’ai pu résoudre sans l’aide de mon maître de stage car aucune solution ne fonctionnait sur Internet.
    Observium n’arrivait pas à récolter les données des différentes machines / équipements et donc n'affichait aucune donnée.
    Une fois l'outil monté avec les machines paramétrées j'ai établi une documentation expliquant la mise en place et la configuration du serveur.
